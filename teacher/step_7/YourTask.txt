Учимся просматривать логи приложения поднятого в Docker контейнере.

Ваша задача: доступ к логам приложения запущенного в Docker контейнере
можно получить следующими способами:

- в приложении Docker Desktop в закладке Containers кликнуть на имя
запущенного контейнера. Откроется окно в котором будут видны логи.

- при запуске контейнера командой из консоли можно указать флаг -i
и тогда логи будут выводиться прямо в консоль:

docker container start insurance-calculator -i

Попробуйте оба способа!

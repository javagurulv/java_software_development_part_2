Ваша задача: создайте Docker container
и научитесь его запускать и останавливать.

Для создания контейнера воспользуйтесь командой:
docker container create
при создании контейнера дайте ему название insurance-calculator
(воспользуйтесь параметром --name).

Для запуска контейнера воспользуйтесь командой:
docker container start.

Для остановки контейнера воспользуйтесь командой:
docker container stop.

Для принудительной остановки контейнера воспользуйтесь командой:
docker container kill.

Для удаления контейнера воспользуйтесь командой:
docker container rm.

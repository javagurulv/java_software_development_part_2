Ваша задача: получить доступ к приложению поднятому в Docker контейнере.
У нашего приложения есть веб часть и REST API. Приложение в дефолтном
варианте поднимается на порту 8080. Для того чтобы можно было
получить доступ к порту контейнера из вне (из нашей основной
операционной системы) при создании контейнера нужно замапить
порт операционной системы на порт контейнера. Это можно сделать
с помощью параметра -p 8080:8080 (первым указывается порт операционной
системы, вторым порт контейнера).

docker container create --name insurance-calculator -p 8080:8080  insurance-calculator:1.0.2

Создайте контейнер с маппингом портов, запустите его,
убедитесь что вы можете в броузере открыть веб UI приложения:
http://localhost:8080/insurance/travel/web/v1
http://localhost:8080/insurance/travel/web/v2
